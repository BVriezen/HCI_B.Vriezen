Computer vision

Processing is een programma waarmee visuele effecten geprogrammeerd kunnen worden. Deze visuele effecten kunnen ook gekoppeld worden aan bepaalde interacties, een voorbeeld hiervan is een vierkant die van kleur verandert afhankelijk van waar de gebruiker zijn of haar muis houdt. Binnen de eerste workshop zijn er door mij een aantal opdrachten in processing gemaakt om kennis te maken met het programma.

Assignment 1

Voor de eerste opdracht moesten er een aantal ellipsen getekend worden van verschillende groottes en kleuren.

Assignment 2

De tweede opdracht was iets complexer in dat er verschillende figuren met verschillende posities gemaakt moesten worden. Hierbij vond ik het lastig om figuren goed te positioneren omdat bepaalde figuren erg veel hoeken en punten hadden, waardoor het soms lastig was om te weten welk punt aangepast moest worden.

Assignment 3

Voor de derde opdracht heb ik een line gemaakt die langer wordt naarmate de muis verder van het beginpunt van de lijn af beweegt. Ook heeft dit invloed op de kleur van het vierkant.

Assignment 4

Bij de vierde opdracht is gebruik gemaakt van een camera en een processing code waarmee verschillende delen van het gezicht kunnen worden geregistreerd. Hierbij moest ik een aanpassing maken waardoor er om het gezicht van de persoon een groen cirkel kwam. In de opdracht die daarna volgde was het de bedoeling om een rode clownsneus op de neus van de persoon te laten verschijnen.

Assignment 5

Voor de laatste opdracht is gebruik gemaakt van een al bestaande code die doormiddel van de camera in een laptop kleuren kan registreren. Bij deze opdracht ben ik er met code niet hellemaal uitgekomen, ik heb dus niet een extra interactie toegevoegd. In plaats daarvan heb ik het uiterlijk van de weergave van de camera veranderd. Eerst heb ik binnen de code de int rangeWidth waarde groter gemaakt waardoor de camera meer kleuren registreert. Vervolgens heb ik bij opencv.useColor(); een nummer toegevoegd waardoor alles een soort psychedelische kleur krijgt. Binnen deze opdracht heb ik niet de code gegeven omdat deze erg lang is en omdat ik slechts twee kleine aanpassingen heb gedaan binnen de code.
