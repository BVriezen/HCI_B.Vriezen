Computer vision

Processing is een programma waarmee visuele effecten geprogrammeerd kunnen worden. Deze visuele effecten kunnen ook gekoppeld worden aan bepaalde interacties, een voorbeeld hiervan is een vierkant die van kleur verandert afhankelijk van waar de gebruiker zijn of haar muis houdt. Binnen de eerste workshop zijn er door mij een aantal opdrachten in processing gemaakt om kennis te maken met het programma.
